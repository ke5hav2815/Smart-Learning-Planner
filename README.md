Smart Learning Planner is a full-stack web application designed to help students manage their studies efficiently. It allows users to create study schedules, organize tasks, track progress, and improve productivity through a clean and responsive interface.

---

## 🚀 Features

- 📅 Create and manage study schedules
- ✅ Add, edit, and delete tasks
- 📊 Track learning progress
- 🔔 Reminder and notification system
- 📚 Subject-wise task organization
- 🌙 Modern and responsive UI
- 🔐 User authentication and login system
- ☁️ Full-stack architecture
